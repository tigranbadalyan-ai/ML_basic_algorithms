# 🧠 ML_basic_algorithms

Welcome to my repository dedicated to mastering the foundations of Machine Learning. This project is a personal journey through the mathematical "engine" of Data Science.

## 🎯 The Philosophy
Instead of just importing libraries, I am deconstructing classical algorithms to understand **how they work under the hood**. My goal is to bridge the gap between abstract mathematical formulas and efficient Python code.

### What I am doing here:
* 🖋 **Deriving Formulas:** Translating calculus and linear algebra from paper to logic.
* 🛠 **Scratch Implementations:** Building models using only **NumPy** to master vectorization.
* ⚖️ **Benchmarking:** Comparing my results against industry standards like **Scikit-Learn**.
* 🔍 **Research & Study:** Exploring edge cases, numerical stability, and optimization techniques.
